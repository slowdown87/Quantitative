## 六、防割韭菜铁律

> ⚠️ 这些话必须记住：

1. **任何承诺"稳赚不赔"的策略都是骗子**
2. **任何收费教你"内幕消息"的都是骗子**
3. **回测赚钱 ≠ 实盘赚钱**（过拟合陷阱）
4. **不要重仓单一股票**（散户最大死法）
5. **不要频繁交易**（手续费会吃掉利润）
6. **量化是工具，不是圣杯**

### 正确心态

- 量化帮助你决策，不是替代你决策
- 小仓位先试，验证后再加码
- 持续学习，尊重市场

- **Jupyter Notebook**（开发环境）

### finshare用法（已验证）

```python
from finshare import get_data_manager

m = get_data_manager()

# 单只股票
s = m.get_snapshot_data('600519')
print(f"贵州茅台: {s.last_price} ({s.change_pct:+.2f}%)")

# 批量股票
r = m.get_batch_snapshots(['600519','000858','300750'])
for code, snap in r.items():
    print(f"{code}: {snap.last_price} ({snap.change_pct:+.2f}%)")
```
