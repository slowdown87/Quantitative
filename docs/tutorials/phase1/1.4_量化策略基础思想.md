## 一、策略是什么？

**策略 = 一套明确的买卖规则**

```
策略 = 在什么条件下，买什么，买多少
        ↓
    IF 条件 THEN 买入/卖出
```

### 策略的三个要素

| 要素 | 说明 | 问题 |
|------|------|------|
| **标的** | 买什么？哪只股票？ | 全市场选还是单一？ |
| **时机** | 什么时候买/卖？ | 开盘买？收盘买？指标信号？ |
| **仓位** | 买多少？ | 全仓？半仓？分批？ |

### 策略的基本框架

```python
# 策略框架示例
class MyStrategy:
    def __init__(self):
        self.position = 0  # 持仓状态

    def on_bar(self, bar):
        """每天收盘后调用"""
        # 1. 计算指标
        ma5 = bar.close.rolling(5).mean()
        ma20 = bar.close.rolling(20).mean()

        # 2. 生成信号
        if ma5 > ma20 and self.position == 0:
            self.buy(bar.close, 100)  # 买入100股
        elif ma5 < ma20 and self.position > 0:
            self.sell(bar.close, self.position)  # 全部卖出

        # 3. 风控检查
        self.check_risk()
```

## 二、三大基础策略

### 2.1 趋势跟踪策略

#### 核心理念

> **"趋势是你的朋友"** —— 涨的时候跟着买，跌的时候跟着卖

```
趋势跟踪：
- 上涨趋势：均线多头排列 → 持有/买入
- 下跌趋势：均线空头排列 → 卖出/空仓
```

#### 趋势跟踪配图

![趋势跟踪示意](../images/11_trend_following.png)

#### 策略逻辑

| 阶段 | 条件 | 操作 |
|------|------|------|
| 入场 | 均线金叉（MA5上穿MA20） | 买入 |
| 持有 | 均线多头排列（MA5 > MA20） | 持有 |
| 出场 | 均线死叉（MA5下穿MA20） | 卖出 |

#### 趋势跟踪代码示例

```python
def trend_following_strategy(df):
    """
    趋势跟踪策略：MA金叉买，死叉卖
    """
    df['MA5'] = df['Close'].rolling(5).mean()
    df['MA20'] = df['Close'].rolling(20).mean()

    # 金叉信号
    df['BuySignal'] = (df['MA5'] > df['MA20']) & \
                      (df['MA5'].shift(1) <= df['MA20'].shift(1))

    # 死叉信号
    df['SellSignal'] = (df['MA5'] < df['MA20']) & \
                       (df['MA5'].shift(1) >= df['MA20'].shift(1))

    return df
```

#### 趋势跟踪优缺点

| 优点 | 缺点 |
|------|------|
| 顺势而为，不逆势 | 会反复被"假突破"打脸 |
| 趋势行情赚大钱 | 震荡行情来回亏 |
| 规则简单，易执行 | 滞后性大，卖点往往不是最高点 |

### 2.2 均值回归策略

#### 核心理念

> **"涨多了会跌，跌多了会涨"** —— 价格围绕价值波动

```
均值回归：
- 价格偏离均线太多 → 迟早要回来 → 反向操作
- KDJ/RSI超卖 → 买入（预期反弹）
- KDJ/RSI超买 → 卖出（预期回调）
```

#### 均值回归配图

![均值回归示意](../images/12_mean_reversion.png)

#### 策略逻辑

| 阶段 | 条件 | 操作 |
|------|------|------|
| 超卖入场 | RSI < 30 或 股价触及布林下轨 | 买入 |
| 回归持有 | RSI回到50附近 | 持有 |
| 超买出场 | RSI > 70 或 股价触及布林上轨 | 卖出 |

#### 均值回归代码示例

```python
def mean_reversion_strategy(df):
    """
    均值回归策略：RSI超卖买，超买卖
    """
    # 计算RSI
    delta = df['Close'].diff()
    gain = delta.where(delta > 0, 0)
    loss = -delta.where(delta < 0, 0)
    avg_gain = gain.rolling(14).mean()
    avg_loss = loss.rolling(14).mean()
    rs = avg_gain / avg_loss
    df['RSI'] = 100 - (100 / (1 + rs))

    # 超卖买入，超买卖
    df['BuySignal'] = df['RSI'] < 30  # 超卖
    df['SellSignal'] = df['RSI'] > 70  # 超买

    return df
```

#### 均值回归优缺点

| 优点 | 缺点 |
|------|------|
| 逆势抄底，买点低 | 可能继续跌，抄底抄在半山腰 |
| 震荡行情效果好 | 趋势行情会被"套死" |
| 止损明确 | 需要耐心等待 |

### 2.3 套利策略

#### 核心理念

> **"利用价差赚钱"** —— 同时买入低的、卖出身的，赚无风险收益

```
套利：
- 同一个东西，两个价格 → 买低卖高
- 相关品种价差偏离正常值 → 买入低的、卖出身的
```

#### 套利类型

| 类型 | 原理 | 举例 |
|------|------|------|
| **跨品种套利** | 相关品种价差回归 | 螺纹钢/热卷套利 |
| **跨月套利** | 同一品种不同月份价差回归 | 原油近月/远月 |
| **ETF套利** | ETF净值 vs 市场价格 | 套利ETF折扣/溢价 |
| **统计套利** | 历史价差规律 | 两只高度相关股票的价差 |

#### 套利配图

![套利策略示意](../images/13_arbitrage.png)

#### 套利代码示例

```python
def statistical_arbitrage(stock_a, stock_b, window=20):
    """
    统计套利：两只高度相关的股票
    - 价差偏离均值 → 买入低的，卖高的
    - 价差回归均值 → 平仓
    """
    spread = stock_a['Close'] - stock_b['Close'] * (stock_a['Close'].mean() / stock_b['Close'].mean())
    spread_mean = spread.rolling(window).mean()
    spread_std = spread.rolling(window).std()

    # 标准化价差
    z_score = (spread - spread_mean) / spread_std

    # 套利信号
    buy_spread = z_score < -2   # 价差过低，买入spread（买A卖B）
    sell_spread = z_score > 2    # 价差过高，卖出spread（卖A买B）
    close_position = abs(z_score) < 0.5  # 价差回归，平仓

    return buy_spread, sell_spread, close_position
```

#### 套利优缺点

| 优点 | 缺点 |
|------|------|
| 理论上的"无风险"收益 | 实际有风险（价差不回归） |
| 收益稳定 | 机会少，利润薄 |
| 不依赖市场方向 | 需要专业知识和工具 |

## 三、三大策略对比

### 3.1 策略对比表

| 对比 | 趋势跟踪 | 均值回归 | 套利 |
|------|---------|---------|------|
| **理念** | 顺势而为 | 物极必反 | 利用价差 |

## 四、散户应该选哪个策略？

### 4.1 根据性格选

| 性格 | 适合策略 |
|------|---------|
| 耐心好，能等待 | 均值回归（RSI超卖买入） |
| 反应快，能止损 | 趋势跟踪（金叉买死叉卖） |
| 数学好，懂相关品种 | 套利（需要计算价差） |

### 4.2 根据资金量选

| 资金量 | 适合策略 |
|--------|---------|
| <10万 | 趋势跟踪、ETF定投 |
| 10-50万 | 趋势跟踪为主，均值回归辅助 |
| >50万 | 可以考虑套利对冲 |

### 4.3 建议

**对于散户，最实用的是：**

1. **趋势跟踪 + 均值回归 组合策略**
   - 趋势明确时：用趋势跟踪
   - 震荡时：用均值回归

2. **简化版**：
   ```python
   # 简化趋势跟踪策略
   if MA5 > MA20 and RSI < 70:
       buy()  # 趋势向上且未超买
   elif MA5 < MA20 or RSI > 80:
       sell()  # 趋势向下或超买
   ```

## 五、策略周期选择

### 5.1 不同周期的特点

| 周期 | 交易频率 | 手续费影响 | 策略稳定性 |
|------|---------|-----------|-----------|
| **高频**（分钟级） | 每天几十上百次 | 影响巨大 | 难稳定 |
| **日内**（日内收盘前） | 每天1-2次 | 中等 | 较难 |
| **日线**（每天收盘后） | 每周几次 | 影响小 | 相对稳定 |
| **周线** | 每月几次 | 几乎无 | 稳定 |

### 5.2 散户建议

> **散户用日线级别就够了** —— 数据免费、交易不频繁、策略稳定

## 六、策略开发流程

### 6.1 完整流程

```
想法 → 策略设计 → 指标计算 → 信号生成 → 回测验证 → 实盘运行
  ↓        ↓           ↓           ↓          ↓          ↓
有灵感   用代码实现   计算MACD    if金叉    跑历史数据  真金白银
```

### 6.2 策略开发检查清单

```
[ ] 策略逻辑清晰，能用一句话描述
[ ] 买卖条件明确，没有歧义
[ ] 有明确的止损/止盈规则
[ ] 回测结果在可接受范围内
[ ] 实盘前做过模拟盘验证
[ ] 知道策略的适用行情和失效行情
```

## 七、常见策略失效原因

### 7.1 过拟合

> **过拟合 = 策略只在历史数据上有效，实盘就亏钱**

| 问题 | 表现 | 解决 |
|------|------|------|
| 参数优化过度 | 某个参数组合回测特别好 | 简化参数，用粗糙参数 |
| 未来函数 | 回测用了未来数据 | 检查数据是否对齐 |
| 选择偏差 | 只选赚钱的股票回测 | 全市场回测 |

### 7.2 市场变化

| 变化 | 影响 |
|------|------|
| 制度变化 | T+1、涨跌停板变化 |
| 风格转换 | 价值/成长风格轮动 |
| 参与者变化 | 量化资金增多，策略有效性下降 |

## 十、参考答案

### 第1题

**趋势跟踪 vs 均值回归的核心区别**：

| 对比 | 趋势跟踪 | 均值回归 |
|------|---------|---------|
| 核心理念 | 顺势而为 | 物极必反 |
| 假设 | 趋势会延续 | 偏离会回归 |
| 操作 | 追涨杀跌 | 高抛低吸 |
| 适用行情 | 单边趋势市 | 震荡市 |

**什么时候用哪个**：
- 大盘有明显趋势（均线多头/空头排列）→ 趋势跟踪
- 大盘震荡（来回拉扯）→ 均值回归

### 第2题

**趋势跟踪在震荡市亏钱的原因**：

```
震荡市：股价来回波动，没有明确方向

     价格
      ↑
      │  ╱╲    ╱╲    ╱╲
      │ ╱  ╲  ╱  ╲  ╱  ╲
      │╱    ╲╱    ╲╱    ╲
      └──────────────────→ 时间

趋势跟踪操作：
- 第1次金叉买入 → 回调卖出 → 亏
- 第2次金叉买入 → 回调卖出 → 亏
- 第3次金叉买入 → 回调卖出 → 亏
- ...
反复被扇脸！
```

### 第3题

**年化100%的策略，你会怎么做？**

> ⚠️ **警惕！年化100%极可能是过拟合！**

正确做法：

1. **怀疑**：先怀疑是不是有问题
2. **检查**：有没有未来函数？参数是不是过度优化？
3. **简化**：用更简单/更粗糙的参数测试
4. **样本外测试**：用没参与回测的数据测试
5. **模拟盘验证**：先跑1-3个月模拟盘
6. **小资金实盘**：先小资金试，验证后再加码

**记住**：回测是历史，未来是未知的。保守一点没坏处。
