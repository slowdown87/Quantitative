## 一、A股交易规则

### 1.1 T+1制度

**当天买的股票，当天不能卖。**

```
买入股票    →    下一个交易日才能卖出
   今天          明天（最早）
```

**举例**：
- 周一买入 → 周二才能卖
- 周五买入 → 周一才能卖

**为什么有这个规则？**
- A股1995年之前也是T+0，后来改成了T+1
- 官方说法：减少频繁交易，防止过度投机
- 实际效果：有争议——支持者说市场更稳，反对者说散户不能及时止损反而亏更多
- 另外：机构可以融券变相实现T+0，散户反而受限
```
> ⚠️ 这是A股和港股、美股的主要区别之一（港股、美股都是T+0）

### 1.2 涨跌停板制度

**股票每天最大涨幅/跌幅有限制。**

| 板块 | 涨跌停限制 |
|------|-----------|
| A股主板/中小板 | ±10% |
| 创业板 | ±20% |
| 科创板 | ±20% |
| ST股票 | ±5% |
| 新股上市首日 | ±44%（但有其他规则） |

**举例**：
- 某主板股票昨天收盘10元，今天最多涨到11元（+10%），最多跌到9元（-10%）
- 某创业板股票昨天收盘50元，今天最多涨到60元（+20%），最多跌到40元（-20%）

**注意**：涨停/跌停时，买卖会变得困难（没有对手盘）

---

### 1.3 交易时间

```
A股交易时间（工作日）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

上午：9:30 - 11:30
下午：13:00 - 15:00

注意：
- 9:15-9:25 是集合竞价时间（可以挂单，但不能成交）
- 15:00后不能交易
- 周末不开市
```

**常见问题**：
- 开盘前15分钟我在干嘛？ → 集合竞价，可以挂单
- 收盘后还能买卖吗？ → 不能

---

### 1.4 交易费用

**买卖股票需要支付的费用：**

| 费用项目 | 收费标准 |
|---------|---------|
| 印花税 | 卖出时收0.1%（国家收） |
| 过户费 | 0.001%（中国结算收） |
| 佣金 | 万1-万3（券商收，可谈） |

**举例**：
```
买入10万元股票：
- 佣金：约10元
- 过户费：约1元

卖出10万元股票：
- 佣金：约10元
- 印花税：约100元
- 过户费：约1元
```

**量化交易要注意**：
- 频繁交易会被手续费吃掉利润
- 策略收益要扣除手续费才是真实收益

## 二、投资品种

### 2.1 股票

**股票 = 上市公司的股份，买股票 = 当股东**

| 特点 | 说明 |
|------|------|
| 收益来源 | 差价 + 分红 |
| 风险 | 高（一天最大亏10%/20%） |
| 门槛 | 最低100股起买（几百元起） |
| 流动性 | 好（随时买卖） |

**A股股票代码规则**：
```
上海A股：600xxx、601xxx、603xxx
深圳A股：000xxx、001xxx
创业板：300xxx
科创板：688xxx
```

**举例**：
- 贵州茅台：600519（上交所主板）
- 宁德时代：300750（深交所创业板）
- 中芯国际：688981（上交所科创板）

---

### 2.2 ETF（交易所交易基金）

**ETF = 可以像股票一样买卖的基金**

简单理解：ETF是一篮子股票的组合。

| 对比 | 股票 | ETF |
|------|------|-----|
| 交易对象 | 单个公司 | 一篮子股票 |
| 风险 | 高（单个公司） | 中（分散风险） |
| 交易费用 | 印花税+佣金 | 同股票 |
| 代表品种 | 贵州茅台 | 沪深300ETF（510300） |

**常见ETF类型**：
```
宽基ETF（跟踪大盘指数）
- 沪深300ETF（510300）- 沪深两市最大的300只股票
- 上证50ETF（510050）- 上海最大的50只股票
- 创业板ETF（159915）- 创业板指数

行业ETF
- 芯片ETF（512760）
- 新能源ETF（515700）

债券ETF、黄金ETF、货币ETF...
```

**散户为什么适合买ETF？**
- 不需要研究个股
- 分散风险
- 不会踩雷退市股

---

### 2.3 期货

**期货 = 约定未来某个时间以某个价格买卖的合约**

| 特点 | 说明 |
|------|------|
| 杠杆 | 高（10倍左右，比如10万可以买100万的合约） |
| 风险 | 极高（可能亏光本金，还可能倒欠） |
| T+0 | 当天可以买卖 |
| 代表品种 | 股指期货（IF、IC、IH）、商品期货 |

**举例**：
```
沪深300股指期货（IF）：
- 现在点位4000点
- 每点300元
- 合约价值 = 4000 × 300 = 120万
- 实际只需保证金 12万（10%保证金）

如果指数涨10%（4000→4400）：
- 盈利 = 400 × 300 = 12万（翻倍！）

如果指数跌10%（4000→3600）：
- 亏损 = 400 × 300 = 12万（亏光保证金！）
```

> ⚠️ 期货风险极高，不建议散户轻易碰

---

### 2.4 期权

**期权 = 花钱买一个买卖的权利**

| 类型 | 说明 |
|------|------|
| 认购期权（Call） | 未来可以以某个价格买入的权利 |
| 认沽期权（Put） | 未来可以以某个价格卖出的权利 |

**举例**：
```
你花1000元买了一个期权：
- 约定价格：茅台2000元
- 期限：1个月

1个月后茅台涨到2500：
- 你用2000元买，转手卖2500，赚钱

1个月后茅台跌到1800：
- 你不买，最多亏1000元期权费
```

> ⚠️ 期权复杂，不建议初学者碰

---

### 2.5 各品种对比

| 品种 | 风险 | 难度 | 适合人群 |
|------|------|------|---------|
| 股票 | 高 | 中 | 有研究能力的人 |
| ETF | 中 | 低 | 普通投资者 |
| 期货 | 极高 | 高 | 专业投资者 |
| 期权 | 高 | 极高 | 专业投资者 |

**散户入门建议**：
- 先从ETF开始（沪深300ETF）
- 有经验后再研究个股
- 期货期权不建议碰

## 三、短线、中线、长线的周期定义

### 3.1 周期划分

| 周期 | 时间范围 | 特点 |
|------|---------|------|
| **短线** | 几天~几周 | 追求快速收益，频繁交易 |
| **中线** | 几周~几个月 | 跟随趋势，波段操作 |
| **长线** | 几个月~几年 | 价值投资，复利增长 |

### 3.2 各周期策略特点

```
短线（几天~几周）
━━━━━━━━━━━━━━━━━━━
- 看技术图形、盘口
- 追热点、题材
- 止损要严格
- 手续费影响大
- 难度：★★★★★

中线（几周~几个月）
━━━━━━━━━━━━━━━━━━━
- 跟趋势、均线
- 看MACD等指标
- 适度持仓
- 难度：★★★☆☆

长线（几个月~几年）
━━━━━━━━━━━━━━━━━━━
- 价值投资
- 关注基本面
- 定投指数ETF
- 复利效应
- 难度：★☆☆☆☆
```

### 3.3 散户怎么选？

| 选择 | 适合人群 | 理由 |
|------|---------|------|
| 长线 | 没时间看盘 | 定投ETF，不用盯盘 |
| 中线 | 有一定经验 | 趋势跟踪，相对稳健 |
| 短线 | 时间充裕、爱折腾 | 工作就是交易，很累 |

**建议**：散户优先考虑 **长线+ETF**，或者 **中线趋势跟踪**

## 四、常见术语

| 术语 | 含义 |
|------|------|
| K线 | 一根柱状图，显示一天的开盘、收盘、最高、最低价 |
| 阳线 | 收盘价 > 开盘价，涨（红色/绿色，看软件设置） |
| 阴线 | 收盘价 < 开盘价，跌 |
| 上影线 | 最高价到收盘/开盘的线 |
| 下影线 | 最低价到开盘/收盘的线 |
| 放量 | 成交量比平时大 |
| 缩量 | 成交量比平时小 |
| 突破 | 股价超过某个关键位置 |
| 回调 | 上涨后回落 |
| 回踩 | 下跌后反弹 |
| 洗盘 | 主力故意打压股价 |
| 套牢 | 买入后亏损 |
| 解套 | 从亏损变回不亏 |

## 五、本节核心记住五点

1. **T+1 = 今天买的股票，明天才能卖**
2. **涨跌停 = A股主板±10%，创业板/科创板±20%**
3. **ETF = 一篮子股票，适合散户入门**
4. **短线难度高，长线难度低，ETF+长线是最稳的**
5. **量化主要用日线数据（收盘价）做策略**

## 六、思考题

1. 你之前炒股时，有没有因为T+1制度吃过亏？
2. 创业板股票一天最大能亏多少？
3. 为什么不建议散户重仓单只股票？
4. 你想做短线、中线还是长线？为什么？

---

## 七、参考答案

### 第1题

**T+1吃过亏的例子**：
- 早盘看到大跌想买，结果下午涨回来了（买不了）
- 早盘涨停想卖，结果下午开板了（后悔卖早了）
- 其实T+1是保护散户的，只是需要适应

### 第2题

**创业板最大亏损**：
- 假设昨天收盘100元
- 今天最多跌到80元（-20%）
- 买1万股 = 亏20万

### 第3题

**不建议重仓单只股票的原因**：
- 个股可能退市（血本无归）
- 个股可能踩雷（业绩造假、造假账）
- 黑天鹅事件无法预测
- 分散持仓可以降低风险

### 第4题

**个人建议**：
- 散户最适合 **长线ETF定投**
- 次选 **中线趋势跟踪**（用量化辅助）
- 短线难度最高，需要大量时间，不建议
