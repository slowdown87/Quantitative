## 一、K线（蜡烛图）

### 1.1 什么是K线

K线，又称蜡烛图，用一根柱子表示一天的开盘、收盘、最高、最低价。

### K线构成

```
        最高价
           │
    ┌──────┴──────┐
    │             │
    │  开盘价─────│────  （上影线）
    │      ■■■■   │────  实体
    │  收盘价─────│────  （下影线）
    │             │
    └──────┬──────┘
           │
        最低价
```

### K线分类

| 类型 | 条件 | 颜色 | 含义 |
|------|------|------|------|
| 阳线 | 收盘 > 开盘 | 红色（涨） | 多方占优 |
| 阴线 | 收盘 < 开盘 | 绿色（跌） | 空方占优 |
| 十字星 | 收盘 ≈ 开盘 | - | 多空僵持 |

## 二、均线（MA）

### 2.1 什么是均线

均线 = 过去N天的平均价格。

| 均线 | 含义 | 用途 |
|------|------|------|
| MA5 | 5日均线 | 短期趋势 |
| MA10 | 10日均线 | 短期趋势 |
| MA20 | 20日均线 | 中期趋势 |
| MA60 | 60日均线 | 长期趋势 |

### 计算公式

```
MA5 = (今日收盘 + 前4日收盘) / 5
MA20 = (今日收盘 + 前19日收盘) / 20
```

### 均线配图

![均线图](../images/02_ma.png)

### 均线使用技巧

| 信号 | 条件 |
|------|------|
| 金叉 | 短期均线从下穿上长期均线 → 买入信号 |
| 死叉 | 短期均线从上穿下长期均线 → 卖出信号 |
| 多头排列 | 短 > 中 > 长均线 → 强势上涨 |
| 空头排列 | 短 < 中 < 长均线 → 弱势下跌 |

## 三、MACD

### 3.1 什么是MACD

MACD（Moving Average Convergence Divergence）是趋势类指标，用于判断股价的趋势方向和动量。

### 三个组成部分

| 组成 | 名称 | 说明 |
|------|------|------|
| DIF | 快线 | 12日EMA - 26日EMA |
| DEA | 慢线 | DIF的9日EMA |
| MACD柱 | 差值柱 | (DIF - DEA) × 2 |

### 计算公式

```python
EMA12 = 12日指数移动平均
EMA26 = 26日指数移动平均
DIF = EMA12 - EMA26
DEA = DIF的9日EMA
MACD = (DIF - DEA) × 2
```

### MACD配图

![MACD图](../images/03_macd.png)

### MACD使用技巧

| 信号 | 条件 | 操作 |
|------|------|------|
| 金叉 | DIF从下穿上DEA | 买入 |
| 死叉 | DIF从上穿下DEA | 卖出 |
| 底背离 | 股价创新低，MACD没创新低 | 潜在买入信号 |
| 顶背离 | 股价创新高，MACD没创新高 | 潜在卖出信号 |

## 四、KDJ

### 4.1 什么是KDJ

KDJ（随机指标）是一种超买超卖指标，用于判断股价的短期强弱。

### 三个组成部分

| 组成 | 范围 | 说明 |
|------|------|------|
| K | 0-100 | 快速随机指标 |
| D | 0-100 | 慢速随机指标 |
| J | 超范围 | K和D的偏离程度 |

### 计算公式

```python
RSV = (今日收盘 - 9日内最低价) / (9日内最高价 - 9日内最低价) × 100
K = RSV的2日指数移动平均
D = K的2日指数移动平均
J = 3 × K - 2 × D
```

### KDJ配图

![KDJ图](../images/04_kdj.png)

### KDJ使用技巧

| 信号 | 条件 | 操作 |
|------|------|------|
| 超买 | K、D、J > 80 | 考虑卖出 |
| 超卖 | K、D、J < 20 | 考虑买入 |
| 金叉 | K从下穿上D | 买入信号 |
| 死叉 | K从上穿下D | 卖出信号 |

## 五、RSI

### 5.1 什么是RSI

RSI（相对强弱指标）衡量股价涨跌的强度，数值在0-100之间。

### 计算公式

```python
上涨平均值 = N日内上涨幅度的平均值
下跌平均值 = N日内下跌幅度的平均值（取绝对值）
RS = 上涨平均值 / 下跌平均值
RSI = 100 - (100 / (1 + RS))
```

### RSI配图

![RSI图](../images/05_rsi.png)

### RSI使用技巧

| 信号 | 条件 | 操作 |
|------|------|------|
| 超买 | RSI > 70 | 考虑卖出 |
| 超卖 | RSI < 30 | 考虑买入 |
| 金叉 | RSI从下穿50线 | 短期转强 |
| 死叉 | RSI从上穿50线 | 短期转弱 |

## 六、布林带（Bollinger Bands）

### 6.1 什么是布林带

布林带由三条线组成：上轨、中轨、下轨，用于判断股价的相对高低。

### 三条轨道

| 轨道 | 计算 | 含义 |
|------|------|------|
| 中轨 | 20日均线 | 股价中期趋势 |
| 上轨 | 中轨 + 2倍标准差 | 股价压力位 |
| 下轨 | 中轨 - 2倍标准差 | 股价支撑位 |

### 计算公式

```python
中轨 = 20日收盘价的均线
标准差 = 20日收盘价的标准差
上轨 = 中轨 + 2 × 标准差
下轨 = 中轨 - 2 × 标准差
```

### 布林带配图

![布林带图](../images/06_bollinger.png)

### 布林带使用技巧

| 信号 | 条件 | 操作 |
|------|------|------|
| 突破上轨 | 股价 > 上轨 | 考虑卖出（超买） |
| 跌破下轨 | 股价 < 下轨 | 考虑买入（超卖） |
| 收口 | 布林带变窄 | 可能有大幅波动 |
| 开口 | 布林带变宽 | 趋势可能加速 |

## 七、指标对比总结

| 指标 | 类型 | 周期 | 适合用途 |
|------|------|------|---------|
| 均线MA | 趋势 | 短/中/长 | 判断方向、金叉死叉 |
| MACD | 趋势+动量 | 中期 | 判断趋势转折 |
| KDJ | 超买超卖 | 短期 | 判断超买超卖 |
| RSI | 超买超卖 | 短期 | 判断涨跌强度 |
| 布林带 | 波动性 | 中期 | 判断支撑压力位 |

## 八、量化中怎么用这些指标？

### 指标在量化策略中的角色

```
指标计算 → 信号生成 → 策略决策
   ↓            ↓           ↓
用代码算     if 金叉      买入/卖出
MACD等      then buy()
```

### 指标代码示例（Python）

```python
import pandas as pd

# 假设 df 是股票数据，有 Close 列

# 1. 计算均线
df['MA5'] = df['Close'].rolling(5).mean()
df['MA20'] = df['Close'].rolling(20).mean()

# 2. 金叉/死叉信号
df['GoldenCross'] = (df['MA5'] > df['MA20']) & (df['MA5'].shift(1) <= df['MA20'].shift(1))
df['DeathCross'] = (df['MA5'] < df['MA20']) & (df['MA5'].shift(1) >= df['MA20'].shift(1))

# 3. 买入信号
buy_signals = df[df['GoldenCross']]
print(f"买入信号数量: {len(buy_signals)}")
```

> ⚠️ 注意：指标有滞后性，信号发出时行情已经走了一段

## 九、常见问题

### Q：指标会不会同时发出矛盾信号？

A：会。比如KDJ显示超买，但均线显示多头排列。这时候要看你的策略周期：
- 短线策略：KDJ优先
- 中长线策略：均线优先

### Q：参数要改吗？

A：默认参数（MA20、MACD 12/26/9、KDJ 9、RSI 14）是经验值，对散户来说够用了。

### Q：是不是指标越多越好？

A：不是。指标太多会互相矛盾，反而不好决策。一般用2-3个就够了。

## 十二、参考答案

### 第1题

**金叉**：短期均线从下穿上长期均线，代表短期上涨动力强 → 买入信号

**死叉**：短期均线从上穿下长期均线，代表短期下跌动力强 → 卖出信号

```
金叉示意：           死叉示意：
    \              /
     \  MA5       /  MA5
      \    \    /       \
       \    \/         \    \
        \   /\         /\    \
  MA20   \ /   \  MA20  \/    \ /
         / \    \       /\    /
        /   \    \     /  \  /
```

### 第2题

| 对比 | MACD金叉 | KDJ金叉 |
|------|---------|---------|
| 周期 | 中期（12/26日） | 短期（9日） |
| 敏感性 | 较慢 | 较快 |
| 适用 | 中长线趋势判断 | 短线买卖点 |
| 频率 | 较少 | 较多（假信号也多） |

### 第3题

**滞后原因**：所有指标都基于历史数据计算，等指标发出信号时，行情已经走了一段。

举例：MA20均线，今天的MA20要用到今天和前19天的数据，本身就有延迟。

### 第4题

**没有标准答案**，取决于你的策略周期：

| 策略 | 处理方式 |
|------|---------|
| 短线策略 | 以KDJ为主，超买就卖 |
| 中长线策略 | 以均线为主，趋势没变就持有 |
| 综合策略 | 均线多头排列时，忽略KDJ超买信号 |

**建议**：提前制定规则并严格执行，不要临时主观判断。
